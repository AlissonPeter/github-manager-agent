GitHub Manager Agent

Agente Inteligente para Gerenciamento de Issues via Terminal Interativo

Aluno: Alisson Peter de Sena

Curso: SCTEC - IA Para DEV's

Problema

Gerenciar manualmente issues no GitHub é trabalhoso e propenso a erros. Criar, editar e fechar issues demanda tempo e segue padrões inconsistentes entre a equipe.

Solução

Agente que interpreta comandos em linguagem natural via terminal, utiliza LLM local (Ollama) para gerar conteúdo profissional e executa ações automaticamente na API do GitHub.

Ferramentas

  • Criação automática de issues com LLM
  • Detecção de duplicatas
  • Notificações via Telegram
  • Interface interativa com preview

Stack Tecnológica

LangGraph Ollama GitHub API Telegram Bot Python

🧠 Processamento com LLM

Ollama local (exemplo: llama3.2:3b) interpreta comandos, gera descrições profissionais e sugere labels automaticamente — sem custos de API externa.

🔄 Fluxo com Validação

Cada etapa valida dados antes de prosseguir: confirmação do usuário, detecção de duplicatas e tratamento de erros da API do GitHub.
